Programmatic access (REST API)

THEOBROMA provides a JSON API for programmatic access. All endpoints return JSON by default.

GET /api/searchSearch compounds by name, SMILES, InChIKey, organism, kingdom, region, or source. Params: q, type, limit, offset, format (json/csv)
GET /api/compound/<id>Full details for a single compound by comp_id
GET /api/autocompleteName autocomplete. Params: q (min 2 chars)
GET /api/organismsOrganism name autocomplete. Params: q
GET /api/statsDatabase summary statistics

Examples

curl "https://theobroma.l3s.uni-hannover.de/api/search?q=curcumin&type=name&limit=10"
curl "https://theobroma.l3s.uni-hannover.de/api/search?q=Streptomyces&type=organism&format=csv"
curl "https://theobroma.l3s.uni-hannover.de/api/compound/THEO_0856795"
curl "https://theobroma.l3s.uni-hannover.de/api/stats"
