Programmatic access

Use the JSON API to query THEOBROMA programmatically:

curl "https://theobroma.l3s.uni-hannover.de/api/search?q=curcumin&type=name&limit=10"

The API returns JSON with fields: comp_id, name, smiles, inchikey, kingdom, source_db, source_organism, region.
