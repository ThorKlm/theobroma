Help

About THEOBROMA

THEOBROMA is an open, curated natural products database aggregating compounds from 27 source databases spanning six biological kingdoms (plant, fungi, bacteria, marine, animal, food) and six continents. It supports structure-based virtual screening, cheminformatics research, and drug discovery from natural product chemical space.

Search

Compound name — partial matching, case-insensitive. Autocomplete suggestions appear after 2 characters. Also searches PubChem synonyms (e.g., "alpha-lipoic acid" finds "Thioctic acid"). Hyphens and Greek letters are normalized automatically.

SMILES — exact canonical SMILES match.

InChIKey — exact 27-character match. If found, redirects directly to the compound page.

Source organism — search by producing organism with autocomplete.

Source database — filter by originating database. Clicking the dropdown shows all 27 available sources.

Geographic region — search by region of origin. Clicking the dropdown shows all 13 resolved regions.

Kingdom — filter by biological kingdom (plant, fungi, bacteria, marine, animal, multi).

Chemical property — filter by chemical class (NP Classifier/ClassyFire) or by numerical property ranges including molecular weight, LogP, TPSA, HBA, HBD, rings, rotatable bonds, and ADMET predictions (solubility, lipophilicity, DILI risk, hepatocyte clearance).

Multi-filter search

Click "+ Add filter" to combine multiple criteria with AND logic. For example, search for all triterpenoids from Africa, or all fungal compounds with molecular weight 200-500 Da and low DILI risk.

Results are relevance-sorted — exact name matches appear first, followed by prefix matches, then substring matches. Within each tier, shorter (simpler) names rank higher.

All result tables have sortable columns — click any column header to sort ascending/descending. Use the "Named only" checkbox in Browse to filter out unnamed compounds. Press / to focus the search box.

Similarity search

Enter a SMILES string, compound name, or THEO_ ID to find structurally similar compounds. Three metrics are available: Morgan/ECFP4 Tanimoto, MACCS keys Tanimoto, and ChemBERTa cosine similarity. Results can be filtered by region, kingdom, and chemical class.

Substructure search

Enter a SMILES or SMARTS pattern to find all compounds containing that substructure. Uses Morgan fingerprint pre-screening followed by RDKit substructure matching.

Scaffold browser

Browse compounds grouped by their Bemis-Murcko molecular framework. Click any scaffold to see all compounds sharing that core structure.

Advanced search

Combine multiple text filters with AND logic and set numerical ranges for physicochemical properties. Property distributions are shown as interactive histograms with selectable ranges.

Provenance

Each compound tracks its full provenance across all source databases. The all_sources field lists every database where the compound was found (pipe-separated). The primary source_db preferentially shows regional/specialized databases over aggregators like COCONUT.

Compounds with "unresolved" region or kingdom lack geographic or taxonomic metadata in their source databases.

How to cite

Klamt, T. et al. (2026). THEOBROMA: an open multi-kingdom natural products database
with per-compound license tiers for legally-aware drug discovery.
Manuscript in preparation.

Please also cite the original source databases for any compounds you use. Source database citations and licenses are listed at /sources.

API

GET /api/search?q=QUERY&type=TYPE&limit=N&offset=M&format=FORMAT

Parameters:
  q      — search query (required)
  type   — name | smiles | inchikey | kingdom | organism | region | source
  limit  — max results, 1-10000 (default: 50)
  offset — pagination offset (default: 0)
  format — json | csv (default: json)

GET /api/compound/COMP_ID     — full compound details
GET /api/autocomplete?q=QUERY — name autocomplete
GET /api/organisms?q=QUERY    — organism autocomplete
GET /api/stats                — database statistics
