About THEOBROMA

THEOBROMA is an open, curated natural products database aggregating compounds from 21+ source databases spanning five biological kingdoms (plant, fungi, bacteria, marine, food) and six continents. It supports structure-based virtual screening, cheminformatics research, and drug discovery from natural product chemical space.

Search

Compound name — partial matching, case-insensitive.

SMILES — exact canonical SMILES match.

InChIKey — exact 27-character InChIKey match.

Source organism — search by the producing organism.

Source database — filter by originating database.

Geographic region — search by region of origin.

Kingdom — filter by biological kingdom.

All result tables have sortable columns — click any column header to sort ascending/descending.

Provenance

Each compound tracks its full provenance across all source databases. The all_sources field lists every database where the compound was found (pipe-separated). The primary source_db preferentially shows regional/specialized databases over aggregators like COCONUT, giving credit to the original curators.

Compounds with "unresolved" region or kingdom lack geographic or taxonomic metadata in their source databases.

How to cite

Klamt, T. et al. (2026). THEOBROMA: an open multi-kingdom natural products database
spanning 21+ sources across five kingdoms and six continents for virtual screening
and drug discovery. Nucleic Acids Research (submitted).

Please also cite the original source databases for any compounds you use.

API

GET /api/search?q=QUERY&type=TYPE&limit=N

Parameters:
  q     — search query (required)
  type  — name | smiles | inchikey | kingdom | organism (default: name)
  limit — max results, 1-1000 (default: 50)

Example:
  curl "https://theobroma.l3s.uni-hannover.de/api/search?q=quercetin&type=name&limit=5"
