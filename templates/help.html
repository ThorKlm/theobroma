About THEOBROMA

THEOBROMA is an open, curated natural products database aggregating compounds from 21+ source databases spanning five biological kingdoms (plant, fungi, bacteria, marine, food) and six continents. It is designed to support structure-based virtual screening, cheminformatics research, and drug discovery from natural product chemical space.

Search

The search interface supports queries by:

Compound name — partial matching, case-insensitive (e.g., curcumin).

SMILES — exact canonical SMILES match.

InChIKey — exact 27-character InChIKey match (e.g., VFLDPWHFBUODDF-FCXRPNKRSA-N).

Source organism — search by the organism that produces the compound (e.g., Curcuma longa).

Source database — filter by the originating database (e.g., IMPPAT, COCONUT).

Geographic region — search by the geographic region of origin (e.g., India, Africa).

Kingdom — filter by biological kingdom (plant, fungi, bacteria, marine, food).

Browse

The browse interface allows filtering the entire compound library by kingdom and/or source database, with paginated results showing key properties for each compound.

How to cite

If you use THEOBROMA in your research, please cite:

Klamt, T. et al. (2026). THEOBROMA: an open global multi-kingdom natural products
database for virtual screening and drug discovery. Nucleic Acids Research (submitted).

Please also cite the original source databases for any compounds you use. The source_db field in each record indicates the originating database.

API

THEOBROMA provides a JSON API at /api/search for programmatic access:

GET /api/search?q=QUERY&type=TYPE&limit=N

Parameters:
  q      — search query (required)
  type   — name | smiles | inchikey | kingdom | organism (default: name)
  limit  — max results, 1-1000 (default: 50)

Example:
  curl "https://theobroma.l3s.uni-hannover.de/api/search?q=quercetin&type=name&limit=5"

License

THEOBROMA operates under a dual-tier licensing model:

Tier 1 — CC BY 4.0: Compounds from open-access sources (COCONUT, MIBiG, NPAtlas, CSIRO, phytochemdb, EMNPD, CyanoMetDB, SWMD, YMDB, AfroDb, SANCDB, ANPDB).

Tier 2 — CC BY-NC 4.0: Compounds from sources with non-commercial restrictions (HERB, Phyto4Health, IMPPAT, MeFSAT, FooDB, VIETHERB).

Each compound record includes a license_tier field. Users performing commercial applications should filter for Tier 1 compounds.

The THEOBROMA web application source code is available under the MIT License on GitHub.
